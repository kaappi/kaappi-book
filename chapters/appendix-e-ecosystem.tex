\chapter{Ecosystem Packages}
\index{ecosystem}\index{thottam!packages}
\label{app:ecosystem}

Kaappi's package manager \texttt{thottam} provides access to a growing ecosystem of libraries. Install any package with:

\begin{Verbatim}[fontsize=\small]
thottam install package-name
\end{Verbatim}

Dependencies are resolved automatically---installing \texttt{kaappi-web} also pulls in \texttt{kaappi-http} and \texttt{kaappi-json}. Packages land in \texttt{\textasciitilde/.kaappi/lib/} and are immediately importable from any Scheme file. All packages are MIT licensed. For the full set of \texttt{thottam} commands (removing, updating, pinning versions), see Appendix~\ref{app:cli}.

\begin{note}[Native Packages Need System Libraries]
Packages that wrap C libraries via the FFI compile a small native shim when installed. OpenSSL is needed by \texttt{kaappi-net} and \texttt{kaappi-crypto}, libpq by \texttt{kaappi-pg}, and libsqlite3 by \texttt{kaappi-sqlite}. The pure-Scheme packages have no system requirements.
\end{note}

\section{Quick Start Examples}

\subsection{Reading JSON}

JSON objects parse to association lists, arrays to lists---look up keys with \code{assoc}:

\begin{lstlisting}
(import (kaappi json))

(define data
  (call-with-input-file "config.json" json-read))
(display (cdr (assoc "name" data)))
\end{lstlisting}

\subsection{HTTP Request}

\begin{lstlisting}
(import (kaappi http))

(define response (http-get "https://api.example.com/data"))
(display (response-body response))
\end{lstlisting}

\subsection{PostgreSQL Query}

Each row comes back as a vector:

\begin{lstlisting}
(import (kaappi pg))

(define conn (pg-connect "host=localhost dbname=mydb"))
(define rows (pg-query conn "SELECT name, age FROM users"))
(for-each (lambda (row)
            (display (vector-ref row 0))
            (newline))
          rows)
(pg-close conn)
\end{lstlisting}

\subsection{CLI Tool}

Flags and options take a short name, a long name, and a description; parsed values are read back by their long name:

\begin{lstlisting}
(import (kaappi cli))

(define app
  (cli "greet" "A greeting tool"
    (flag "-l" "--loud" "Use uppercase")
    (option "-n" "--times" "Repeat N times" 1)
    (argument "name" "Name to greet")))

(define result (run-cli-parse app '("-l" "Ada")))
(parsed-flag? result "loud")                ; => #t
(cdr (assoc "name" (parsed-args result)))   ; => "Ada"
(parsed-ref result "times")                 ; => 1 (default)
\end{lstlisting}
