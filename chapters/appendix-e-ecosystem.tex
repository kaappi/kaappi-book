\chapter{Ecosystem Packages}
\index{ecosystem}\index{thottam!packages}

Kaappi's package manager \texttt{thottam} provides access to a growing ecosystem of libraries. Install any package with:

\begin{Verbatim}[fontsize=\small]
thottam install package-name
\end{Verbatim}

After installation, import the library in your Scheme code. All packages are MIT licensed.

\section{Quick Start Examples}

\subsection{Reading JSON}

\begin{lstlisting}
(import (kaappi json))

(define data (json-read (open-input-file "config.json")))
(display (hash-table-ref data "name"))
\end{lstlisting}

\subsection{HTTP Request}

\begin{lstlisting}
(import (kaappi http))

(define response (http-get "https://api.example.com/data"))
(display (response-body response))
\end{lstlisting}

\subsection{PostgreSQL Query}

\begin{lstlisting}
(import (kaappi pg))

(define conn (pg-connect "host=localhost dbname=mydb"))
(define rows (pg-query conn "SELECT name, age FROM users"))
(for-each (lambda (row)
            (display (vector-ref row 0))
            (newline))
          rows)
(pg-close conn)
\end{lstlisting}

\subsection{CLI Tool}

\begin{lstlisting}
(import (kaappi cli))

(define app
  (cli "mytool"
    (version "1.0.0")
    (description "A helpful tool")
    (flag "verbose" "v" "Enable verbose output")
    (argument "file" "Input file")))

(define opts (cli-parse app (command-line)))
\end{lstlisting}

\section{Managing Packages}

\begin{center}
\begin{tabular}{lp{7cm}}
\textbf{Command} & \textbf{Description} \\
\hline
\texttt{thottam install pkg} & Install a package \\
\texttt{thottam remove pkg} & Remove a package \\
\texttt{thottam list} & List installed packages \\
\texttt{thottam update} & Update all packages to latest versions \\
\texttt{thottam verify} & Verify integrity of installed packages \\
\texttt{thottam search term} & Search for packages by name \\
\end{tabular}
\end{center}

Packages are installed to \texttt{\textasciitilde/.kaappi/lib/} and are immediately importable from any Scheme file.
